project: SG HDB lifts dimension collector

specs:
- tech stack
    1. frontend: nextjs
    2. backend: nestjs
    3. database: sqlite
    4. manage both frontend and backend using turborepo

- features
    1. simple CRUD operations in form
    2. Single page application (mobile friendly website)
    3. generate csv based on input data
    4. preview before generating csv
    5. get the full Singapore address based on the HDB block numbers
    6. simple register/login account feature
